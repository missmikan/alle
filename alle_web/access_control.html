設計理念 Design Concept
專為校園設計的門禁系統，包含管理校園的出入紀錄、自動開門管理、電梯使用權限等等，讓學校師生皆可一卡通行，也能有效管理校內的所有教室、辦公室與各大場地。更提供客製化設計製卡、門禁設備安裝、門禁與系統整合應用、各縣市機關之卡片管理系統平台設計開發等服務。
 使用控制轉換器，將各處室刷卡資料與電腦系統設定權限相互傳送。
應用功能 System Features

課表結合

連結校務系統的課表，幫老師事前預約好課表上的教室，當天即可持數位證開啟教室。

E卡通行

可支援多元卡種，數位教師證能通行辦公室、專科教室、校門、電梯等等

校園安全

凡走過必留下痕跡，只要感應過卡片，系統將會自動記錄使用時間，校園安全可掌握。

電梯管制

可設定搭乘師生或傷病學生臨時搭乘，有效管控電梯使用量以及電量的使用，有效節能環保

資安保障

若貴校有教師或學生異動離校，該師生的數位卡將會自動停用，通行權自動關閉。

即時APP

與校務APP整合，連結場地預約管理、出入管理，還可用APP遠端開門。
軟體功能 Software Features
此圖為刷卡紀錄系統
場地預約
  • 場地預約借用 線上登記預約使用電梯或場地
  • 通行記錄查詢 可隨時查看通行記錄
  • 永久通行設定 可設定永久通行人員
單位人事製卡
  • 手動開卡/停卡 通行卡片遺失時可進行卡片補發
  • 卡片補登 將新辦的卡片資料匯入系統
請假管理
  • 差勤記錄 藉由卡片感應完成上下班打卡或點名
校園出入管理
  • 學生出入紀錄 可按學生姓名、座號或學號查詢進出刷卡記錄
  • 上下學時間設定 可依班級類別設定上下學時間

設計理念 Design Concept
提供給校護的健康掛號解決方案—健康掛號系統，提供手機APP與健康掛號機兩種方式，可以清楚瀏覽日掛號人次、歷次掛號紀錄。學生透過系統完成掛號，校護完成處理後，快速紀錄於系統，並發送推播通知，讓導師、家長皆能第一時間掌握學生傷病情形。此外，還可透過大數據分析許多面向，例如：受傷地點，了解校內哪些設施讓孩子容易受傷，儘快改善便得以提升校園安全。
系統特色 System Features

歷次掛號紀錄

記錄學生歷次傷病紀錄與處理方式。

統計人數

顯示當日掛號人次、休息觀察人數，校護輕鬆掌握保健室情形，導師清楚得知班級狀況。

即時通知家長

校護填寫醫囑後，推播至家長的手機APP內，家長將會立即收到通知。

兩種介面選擇

擁有平板與手機兩種使用介面選擇，讓學校挑選最適用的介面。

傳送至健康系統

學生的掛號資訊將同步傳送至教育部學生健康資訊系統。
手機APP介面 APP Interface

無需另購設備。

  1. 掛號畫面
    首頁統計等待、就診、休息觀察人數。 掛號資料僅需點擊選項，方便快速紀錄各學生掛號資訊。
掛號畫面
首頁統計等待、就診、休息觀察人數。 掛號資料僅需點擊選項，方便快速紀錄各學生掛號資訊。
填寫醫囑
若需注意叮嚀事項，校護可選擇填寫醫囑給家長。 家長觀看後點擊我知道了，將會記錄已讀在該醫囑上。
推播通知至家長
填寫醫囑將會推播至家長的訊息列表內。 家長僅需點擊即可連接至該醫囑。
掛號紀錄
導師觀看班級掛號紀錄。 家長觀看孩子的歷次掛號紀錄。
健康掛號機介面 Registration Machine

需另購健康掛號機。

校護端畫面

當日就診紀錄及待診名單統計。

可立即聯繫學生緊急聯絡人。

若有APP系統，填寫醫囑可選擇推播家長、導師或管理端。

掛號流程

1.點擊我要掛號

2.輸入年班座號或感應數位學生證

3.選擇內外科

4.填寫受傷登記

5.選擇受傷部位

6.確認掛號資訊並完成
